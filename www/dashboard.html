Welcome to Computer Applications Dashboard

This project is about the content management mechanism built using a material design approach for the department itself i.e. Department of Computer Applications, Chitkara University. We are mainly focusing on the languages which are HTML for providing the basic structure to the Portal, jQuery for giving the application the runtime changes, CSS for the design of the application and jQuery AJAX and PHP will be used for database connectivity.

What is our Proposed System?

This project is brought forward to make the departmentâ€™s web portal dynamic and allow the end user to easily manage the content without even interfering with the backend code used to make main interface. The website will be dynamic and will be enhanced periodically. It will be online and will be in alpha stage once the basic structure is acquired. The concept of linking our modular project to any kind of object present in the college.

Scope of Project

This will make the department portal dynamic as well as it will enable the department members to control the content of the main interface from the admin panel. An interaction between the department authorities will help in improving and scaling the imaginable growth for the Department of Computer Application, Chitkara University.

Features.

  • Dynamic Website in which content can be managed without interfering with the backend code.
  • Separate and secured login for the Administrator.
  • Administrator can further add users for the content management purpose.
  • Addition/Deletion of the available navigation bars.
